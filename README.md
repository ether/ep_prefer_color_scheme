# Make  Etherpad react to your systems default settings


For dark mode the following settings apply: 

```javascript
  const nightColors = {
    toolbar: 'dark',
    background: 'super-dark',
    editor: 'dark',
  }
```

For light mode: 

```javascript
  const dayColors = {
    toolbar: 'super-light',
    background: 'light',
    editor: 'super-light',
  }
```
